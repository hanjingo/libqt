### Dependencies

| compiler | version         |
| -------- | --------------- |

    en: "Project"->"Generate Cache"

    zh: "项目"->"生成缓存"

- cmd (reference from: https://learn.microsoft.com/zh-cn/vcpkg/consume/boost-versions):

    ```cmd
    vcpkg install
    ```

or install 3rd-party project by yourself.
